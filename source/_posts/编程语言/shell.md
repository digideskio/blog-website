# 变量
bash变量分为
* 自定义变量
* 环境变量
* 预定义变量


## 变量声明
```shell
v1=123
```
在上面的声明语法中我们需要注意以下几点
* `=`左右不能用空格
* 变量的默认类型是字符串
* 该变量对当前以及子shell都有效

### declare声明
```
declare v1=
```

## 变量引用
我们通过使用`$`或者`${}`符号可以引用一个变量
```shell
v=1
echo $v
echo ${v}
```
